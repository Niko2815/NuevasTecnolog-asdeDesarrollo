PASOS PARA LA EJECUCION DEL CODIGO (NUEVAS_TECNOLOGIAS_DE_DESARROLLO).

Paso 1:
Crear una carpeta para abrir en visual studio code (Preferible con el nombre mencionado como titulo)
Paso 2:
Descargar todos los archivos depositados en este repositorio
Paso 3:
Abrir visual estudio -> abrir carpeta (Selecciona la carpeta creada previamente en el paso 1)
Paso 4: 
Crear entorno virtual. Para ello abrimos la terminal (crtl + Ã±), una vez abierta escribimos el siguiente codigo "Python -m venv .venv" y damos enter. Se debe crear una carpeta de nombre ".venv"
Paso 5:
Iniciar entorno virtual. En la misma terminal ejecutamos .\.venv\Scripts\Activate.
Paso 6:
Los archivos previamente descargados los abrimos desde visual studio (Archivo->Abrir archivo-> nombre del archivo-> doble click) esto lo hacemos con to-do.py, app.py y requeriments.txt
Paso 7:
Una vez abierto todos los archivos previamente mencionados creamos una nueva carpeta de nombre "templates"
Paso 8:
Dentro de la carpeta templates abrimos el archivo faltante "index.html" de la misma forma que abrimos los archivos anteriores.
Paso 9:
En la terminal escribimos " python.\app.py " y damos enter. La terminal nos dara un link.
Paso Final:
Abrimos el link dando click sobre el mientras que al tiempo oprimimos las tecla ctrl. Se abrira el navegador y editaremos la url colocando "\inicio" al final de la misma.
